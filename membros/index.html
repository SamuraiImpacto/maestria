Como instalar

  1. Instale o Claude Code no seu computador (uma vez só). Guia passo a passo aqui, feito pra quem nunca mexeu com isso.
  2. Baixe a skill no botão acima e arraste o arquivo ZIP pra conversa do Claude Code com a frase: "instala essa skill pra mim". Ele instala sozinho.
  3. Ative sua licença: digite /maestria ativar, informe seu código de ativação e o CPF do titular. As demais skills do seu plano são oferecidas na sequência, automaticamente.
