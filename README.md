# MaestrIA — Landing

## Estrutura

- `index.html` — landing principal (hero, pacotes, calculadora ROI, FAQ)
- `obrigado.html` — pós-compra Kiwify (4 passos de instalação)
- `suporte.html` — 3 canais + 8 FAQs
- `painel-curadoria.html` — preview do painel pro Banco Vivo

- `calculadora.js` — calc ROI interativa (15 skills)
